# ^oOC||^ GAMING PLAY FULL - Os Causadores

Site oficial do clã **Os Causadores**, nascido na Zona Leste de São Paulo.  
Aqui você encontra o pôster épico, os membros e a história completa do clã.

---

## 🚀 Estrutura do projeto
- `index.html` → Página inicial com pôster e menu lateral
- `membros.html` → Lista completa dos integrantes com imagens e descrições
- `historia.html` → História do clã desde 2008
- `style.css` → Arquivo de estilos para visual moderno e profissional
- `/imagens` → Pasta com todas as artes dos personagens e pôster

## 🌐 Publicação no GitHub Pages
1. Crie um repositório público no GitHub chamado `OC_GAMING_PLAY_FULL`.
2. Faça upload de todos os arquivos.
3. Vá em **Settings > Pages**.
4. Em **Source**, selecione a branch `main` e a pasta `/root`.
5. Clique em **Save**.
6. O site ficará disponível em:  
   `https://seuusuario.github.io/OC_GAMING_PLAY_FULL`
